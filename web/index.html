The first WebAssembly GitHub Actions runner. run: steps execute inside Wasm — right here in this tab. Source on GitHub →

Two modes. Playground (below) runs a workflow locally in this tab — nothing leaves your machine. Connect (right here) registers this tab as a real self-hosted runner: paste your org URL and a registration token, and GitHub will dispatch real jobs to your browser, with run: steps executing in Wasm in-tab.

Connect this tab as a runner

Open Settings → Actions → Runners → New runner, copy the one-click ./config.sh command, and paste it below. Nothing leaves this tab. Workflows target it with runs-on: tabrunner.
