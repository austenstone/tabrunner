⚡tabrunner

The first WebAssembly GitHub Actions runner. run: steps execute inside Wasm — right here in this tab. Source on GitHub →

This is an in-tab playground: it runs the workflow locally in your browser using a tiny Wasm shell (no network, no real OS, nothing leaves your machine). It does not register as a runner that GitHub dispatches jobs to — that needs a persistent backend connection. Think of it as "what if a browser tab could execute a workflow."

Built with wazero + a writable in-memory filesystem, compiled to js/wasm. Steps run in a hand-written WASI shell embedded in the page. austenstone/tabrunner
